# 🔥 Ballistics Reforged

**Ballistics Reforged** adds extra projectile feel, battlefield feedback, and visual polish to Factorio combat so bullets, rockets, shells, shotguns, turrets, and spitter shots feel more physical.

Instead of combat feeling completely instant and static, weapons now have visible tracers, cleaner impact feedback, smoke trails, arcing ordnance, and better battlefield readability.

Shoot cleaner. Hit harder. Make combat feel alive without turning every fight into a messy effects cloud.

## ⚙️ What It Does

Ballistics Reforged gives Factorio combat a stronger ballistic feel.

Bullets can become visible physical tracers. Rockets can feel more like fired ordnance instead of perfect homing missiles. Shotguns can feel punchier and more controlled. Cannon shells, grenades, capsules, and atomic bombs can travel in visible arcs.

Impacts also get more visual life, with optional dust, debris, bug-hit feedback, structure-hit feedback, rocket smoke, cannon trails, and spitter impact effects.

The goal is not to turn Factorio into a different game. The goal is to make combat feel sharper, heavier, and more believable while still fitting the normal Factorio style.

This Factorio 2.0 version includes cleaner prototype handling, improved settings, FX presets, impact layering, big-fight tuning, and safer compatibility options.

## ✅ Features

### 🔫 Ballistics and Projectile Feel

* Physical bullet tracers
* Optional glowing bullet trails
* Optional bullet smoke trails
* Configurable bullet speed, range, and inaccuracy
* Tuned shotgun pellet behaviour
* Configurable shotgun range and fire-rate multiplier
* Physical rockets with configurable speed and wobble
* Rocket smoke wake effects
* Arcing grenades, capsules, cannon shells, and atomic bombs
* Ammo-style visual identity where supported

### 💥 Impact and Battlefield FX

* Bullet impact smoke
* Shotgun impact smoke
* Ground impact dust
* Structure-hit smoke
* Enemy hit feedback
* Rocket wake smoke
* Cannon shell smoke trails
* Heavy ordnance impact FX
* Spitter acid impact FX
* Cleaner large-battle effect control

### 🛡️ Combat Behaviour

* Optional wall-blocked spitter projectiles
* Turret projectile leading
* Better projectile collision feel
* Cleaner big-fight FX cap
* Works with vanilla combat and compatible modded ammo/turrets

## 🛠️ Settings

Ballistics Reforged includes startup settings so you can tune combat feel, visuals, and performance.

| Setting                  | What It Does                                                  |
| ------------------------ | ------------------------------------------------------------- |
| **Combat FX Preset**     | Controls overall FX intensity: Subtle, Balanced, or Cinematic |
| **Impact Layering**      | Controls how many extra impact layers are added               |
| **Big-fight FX Cap**     | Keeps large turret battles cleaner                            |
| **Physical Bullets**     | Enables visible bullet projectiles                            |
| **Bullet Glow**          | Adds glow to bullet tracers                                   |
| **Bullet Smoke Trails**  | Adds subtle smoke behind bullets                              |
| **Shotgun Rework**       | Improves shotgun pellet behaviour                             |
| **Physical Rockets**     | Makes rockets feel more like fired ordnance                   |
| **Rocket Smoke Trails**  | Adds smoke wake to rockets                                    |
| **Arcing Cannon Shells** | Makes cannon shells travel in a visible arc                   |
| **Walls Block Spitters** | Converts spitter streams into wall-blockable projectiles      |
| **Turret Leading**       | Helps turrets lead moving targets                             |

## ⭐ Recommended Settings

For the best clean combat feel:

| Setting                   | Recommended Value |
| ------------------------- | ----------------- |
| **Combat FX Preset**      | Balanced          |
| **Impact Layering**       | Balanced          |
| **Big-fight FX Cap**      | 2                 |
| **Physical Bullets**      | On                |
| **Bullet Glow**           | On                |
| **Rocket Smoke Trails**   | On                |
| **Wall-blocked Spitters** | On                |
| **Turret Leading**        | On                |

This gives a clean, active, and readable combat feel without overloading big fights.

For screenshots or cinematic combat:

| Setting                 | Recommended Value |
| ----------------------- | ----------------- |
| **Combat FX Preset**    | Cinematic         |
| **Impact Layering**     | High              |
| **Big-fight FX Cap**    | 3 or higher       |
| **Bullet Glow**         | On                |
| **Bullet Smoke Trails** | On                |
| **Rocket Smoke Trails** | On                |

For huge bases or heavy turret walls:

| Setting                 | Recommended Value |
| ----------------------- | ----------------- |
| **Combat FX Preset**    | Subtle            |
| **Impact Layering**     | Low               |
| **Big-fight FX Cap**    | 1 or 2            |
| **Bullet Smoke Trails** | Off or Low        |
| **Rocket Smoke Trails** | On                |

## 🧪 Behaviour Notes

* This mod is focused on combat feel and battlefield feedback.
* It does not try to completely rebalance Factorio combat.
* FX presets let you choose between cleaner performance-friendly visuals and heavier cinematic combat.
* Subtle mode is best for large bases and heavy turret walls.
* Balanced mode is the recommended default.
* Cinematic mode is best for screenshots, smaller fights, or players who want more visual punch.
* No source-side muzzle smoke is used, because that can appear at the player’s feet instead of the weapon position.
* The mod is designed to stay lightweight and practical for normal gameplay.

---

## 🧩 Compatibility Notes

Ballistics Reforged edits combat prototypes during the `data-final-fixes` stage so it can affect more vanilla and compatible modded ammo/projectiles.

If another combat overhaul changes the same weapons, projectiles, or ammo categories, use the startup settings to disable overlapping features.

Optional compatibility targets include:

* Rampant Arsenal
* Bob’s Warfare
* Combat Robots Overhaul
