# Ballistics Reforged

Ballistics Reforged is a rebuilt ballistics and battlefield-feel mod for Factorio 2.0 focused on bullets, rockets, shells, tracers, battlefield feedback, and cleaner large-fight visuals.

## Features

- Physical bullet tracers
- Subtle / Balanced / Cinematic FX presets
- Impact layering: Clean / Balanced / Full
- Big-fight FX cap for cleaner large turret battles
- Tuned shotgun behaviour
- Physical rockets with smoke and heavier impact feel
- Arcing grenades, capsules, cannon shells, and atomic bomb
- Optional wall-blocked spitter projectiles
- Turret projectile leading
- Cleaner impact, dust, debris, and bug-hit feedback

## Current version focus

Version 1.0.0 is the first stable public release. It includes the Combat Polish + FX Balance pass, the 288x288 Ballistics Reforged thumbnail, and the final Ballistics Reforged branding.

## Compatibility

This mod edits combat prototypes during data-final-fixes so it can affect more compatible vanilla and modded ammo/projectiles. If another combat overhaul changes the same prototypes, use the startup settings to disable overlapping features.

Optional compatibility targets are listed for Rampant Arsenal, Bob's Warfare, and Combat Robots Overhaul.
